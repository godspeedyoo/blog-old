Bootstrap

After doing some research, I chose to go ahead and implement bootstrap on my blog. Judging by the new technologies to learn and the setup time, I reserved a good chunk of time to go through this and learn how to set it up. Overall, I'm quite pleased with the results, although I think it did take longer than anticipated. Setting up bootstrap and trying to understand what it does took quite a bit of time, and I'm still learning bits and pieces at a time.

Once you have the files set up, you can implement bootstrap by applying many of its provided classes to your HTML. All of the fancy CSS is already compiled, and what you end up with is a library to use many cool classes and helper classes that make life easy for web design. One thing I picked up on here was applying many combinations of classes to a single HTML tag. Having specific classes defined for its unique purpose allows you to flexibly use and apply many combinations of things, such as "btn" and "pull-right" classes to specify a button that you wish to be placed on the right side.

Bootstrap Grids

One of the big concepts I picked up on was the use of grids. Imagine this page separated into 12 distinct columns, and you can use column numbers to specify how much space you want things to take. Working with numbers 1-12 is much easier than using precise pixels. You can find a guide full of examples here.

Design

One of the most powerful things about bootstrap is its ability to make something look pretty with relative ease. Imagine writing your own css to create this button.

Not only does it have built in gradient shading but it comes with the click and hover design. Having generic things like buttons already built in and coded helps us focus on other things, as these can be reused and tweaked to customize our own look and feel.

As I merely implement a very standard and template like look, I plan to update the site with more customization as I mess with my own variables and color schemes. It's merely a matter of time, and I've almost run out. DBC Phase 1 will begin tomorrow!
