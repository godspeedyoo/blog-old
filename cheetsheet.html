Cheatsheet: Looping in Ruby vs. JavaScript

Joseph Won & Nell Taylor

Ruby Examples

Ruby and JavaScript don't have 1:1 comparisons of everything. We'll show you Ruby loops first, then JavaScript

Ruby: For Loops

Input

for i in 0..5
  puts i
end

Output

0
1
2
3
4
5

Ruby: While Loop

Input

count = 0

while count < 5 do
    puts "continuing program.."
    count += 1
end

Output

continuing program..
continuing program..
continuing program..
continuing program..
continuing program..

Ruby: Until Loop with Begin/End

Input

stars = 0

begin

  puts "counting the stars.."
  stars+=1

end until stars == 5

Output

counting the stars..
counting the stars..
counting the stars..
counting the stars..
counting the stars..

Ruby: Using the .each method

Input

(0..5).each do |i|
  puts i
end

Output

0
1
2
3
4
5

JavaScript: For Loop

Input

for (i = 0; i<6; i++) {
  console.log(i);
}

Output

0
1
2
3
4
5

JavaScript: For Loop -- multiple first statements

Input

for ( i = 0, word="hello", len =word.length; i<len; i++ ) {
  console.log(word.charAt(i))
}

l
o
